# margin-calculator
Калькулятор маржи для логистов - это отличный инструмент для работы логистов, который позволяет унифицировать согласования тарифа с руководителем и сократить время на расчеты.
Реализовано мини приложение в Telegram. Для этого просто добавьте бота @margin_calculator_bot в рабочую группу или канал и введите **/start** или нажмите кнопку <img width="662" height="65" alt="image" src="https://github.com/user-attachments/assets/616734e7-6ab6-4a5d-8e94-40504d8dc336" />
